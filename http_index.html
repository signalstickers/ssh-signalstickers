ssh.signalstickers.com

This server hosts a SSH server on port 22, allowing to use signalstickers.com in a terminal.

On *nix

Simply do 

            ssh user@ssh.signalstickers.com
Accept the finderprint, and enter the password that the server will give you (yay, security!)

On Windows

Install PuTTY, and connect to ssh.signalstickers.com with user user. Then, enter the password that the server will give you. If you have WSL installed, refer to the *nix previous chapter.
